简单总结:
- 基本选择器
  - $("元素的名称");
  - $("#id的属性值");
  - $(".class的属性值");
- 层级选择器
  - $("A B");
  - $("A > B");
- 属性选择器
  - $("A[属性名]");
  - $("A[属性名=属性值]");
- 过滤器选择器
  - $("A:even");
  - $("A:odd");
- 表单属性选择器
  - $("A:disabled");
  - $("A:checked");
  - $("A:selected");
